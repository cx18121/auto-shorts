   - Falls back to ("reject", "batch parse error") for any item that can't be parsed

   - Convert all items to dicts: `item_dicts = [dict(item) for item in items]`
   - Call `_ai_review_batch(item_dicts, source_label, channel_cfg)` once
   - Iterate results alongside items to apply approve/reject as before
   - Print each decision as it processes (same output format as now)

3. Keep `_ai_review_item` for now (no deletion) in case single-item review is needed later, but the `cmd_review` AI path must use the batch function.

Cap batch size at 50 items per call. If more than 50 pending, chunk into batches of 50 and make one call per chunk. Truncate each item's content to 400 chars in the batch prompt to stay within context limits.
