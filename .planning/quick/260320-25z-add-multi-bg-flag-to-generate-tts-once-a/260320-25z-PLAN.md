Reuse the same clip-listing logic as `_pick_background_interactive()` (enumerate clips from assets/backgrounds/, show numbered list). Prompt: "Select backgrounds (comma-separated, e.g. 1,3,5): ". Parse comma-separated input. Validate each number is in range. Must select >= 2 backgrounds; if only 1 selected, print "Use --background for single selection" and sys.exit(1). Return list of path strings in selection order.

5. In `_generate_storytelling_from_backlog`, add `backgrounds: list[str] | None = None` parameter. When `backgrounds` is provided:
   - The outer loop still pulls ONE backlog item per iteration (count is still respected as number of stories, not videos).
   - After generating the story via `_generate_with_quality`, instead of calling `_run_storytelling_pipeline` once, call a new helper `_assemble_multi_bg` that:
     a. Creates ONE timestamped workdir.
     b. Generates TTS ONCE into that workdir (narration.mp3, timestamps.json).
     c. Generates subtitles ONCE (subtitles.ass).
     d. Loops over each background in `backgrounds`, calling `assemble_video()` directly (from formats.storytelling.assembler) with output_path = workdir / f"final_bg{idx}.mp4" (1-indexed).
     e. Calls `_save_video_metadata` for each output video.
     f. Returns list of all produced video paths.
   - The backlog item is marked used ONCE (not per background).
   - All produced paths are appended to the `produced` list.
   - Background logging: call `log_background_use` for each background used.

6. Create `_assemble_multi_bg(story_text: str, backgrounds: list[str], no_audio: bool, channel_cfg) -> list[str]` helper in commands/generate.py:
   - Import `generate_tts` from pipeline.tts, `generate_ass` from pipeline.overlay, `assemble_video` and `AUDIO_SPEED` from formats.storytelling.assembler.
   - Create workdir with `int(time.time())` run_id.
   - Generate TTS (or silent audio if no_audio).
   - Generate ASS subtitles.
   - For each background (enumerate starting at 1): call `assemble_video(background_path=bg, audio_path=..., subtitles_path=..., output_path=workdir/f"final_bg{i}.mp4", duration_seconds=...)`.
   - Return list of successful output paths.
   - On per-background assembly failure, log error and continue to next background.
