    1. Extracts a fallback description: take the first two sentences from content_text by splitting on ". " — join the first two elements back with ". " and append a period if it does not already end with one. Cap at 200 chars. Strip leading/trailing whitespace.
    2. Extracts a fallback title: take content_text[:80], strip trailing whitespace and trailing punctuation chars (commas, dashes, spaces) that indicate a mid-word cut, cap cleanly. Do NOT cut mid-word — find the last space within the 80-char window and truncate there. Append "..." if truncated.

    Implement this as a small inline helper (no new function needed — just compute fallback_title and fallback_description in the except block before assigning ai_result):

    ```python
    except Exception as exc:
        logger.error("generate_upload_metadata: Claude call failed: %s", exc)
        # Fallback title: truncate at last word boundary within 80 chars
        _t = content_text.strip()
        if len(_t) > 80:
            _t = _t[:80]
            last_space = _t.rfind(" ")
            if last_space > 40:
                _t = _t[:last_space]
            _t = _t.rstrip(",-") + "..."
        # Fallback description: first 2 sentences, up to 200 chars
        _sentences = [s.strip() for s in content_text.split(". ") if s.strip()]
        _desc = ". ".join(_sentences[:2])
        if _desc and not _desc.endswith("."):
            _desc += "."
        if len(_desc) > 200:
            _desc = _desc[:200].rstrip() + "..."
        ai_result = {"title": _t, "description": _desc, "hashtags": []}
    ```
